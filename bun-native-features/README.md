# Bun Native Features Showcase

## What's Built-in?

- **HTTP Server**: `bun run http-server.js` - No express needed!
- **Testing**: `bun test` - Built-in test runner (like Jest)
- **TypeScript**: Run `.ts` files directly - No config!
- **SQLite**: `bun run sqlite-demo.js` - Native database support
- **Watch Mode**: `bun --watch server.js` - Auto-restart on changes

## Quick Commands

```bash
# Run HTTP server
bun run http-server.js

# Run tests
bun test

# Run TypeScript (no config!)
bun run ts-demo.ts

# SQLite demo
bun run sqlite-demo.js

# Auto-restart on file changes
bun --watch http-server.js
```
