# Node.js Example - External Libraries Required

This shows how Node.js handles common tasks with external packages.

## Install Dependencies

```bash
npm install
# Takes 30-120+ seconds
```

## Required Packages (26 total!)

| Package | Purpose |
|---------|---------|
| express | HTTP server |
| helmet | Security headers |
| cors | CORS support |
| morgan | HTTP logging |
| body-parser | Parse JSON |
| compression | Gzip responses |
| lodash | Utility functions |
| moment | Date handling |
| axios | HTTP client |
| joi | Validation |
| winston | Logging |
| dotenv | Environment vars |
| jest | Testing |
| nodemon | File watching |
| + 13 more... |

## Run

```bash
node server.js
# or
npm run dev
```
